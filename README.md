# Trackify 💰

A simple expense tracking web app I built to manage daily spending and understand where my money goes.

## Features
- Add and store expenses
- Delete expenses
- View total spending
- Category-based analytics (pie chart)
- Clean and minimal UI

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS
- Chart.js
